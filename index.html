USPTO Chemical Compounds Extraction Proof-of-concept

Extracted chemical compounds from USPTO CDX attachments (2001–2025), focussed on pharma‑related IPC/CPC classes C07, A61K, A61P, A01N. Molecules are serialised as CXSMILES with RDKit.

Browse index (contents.json)

Each weekly USPTO bulk release has a JSON listing patents/applications that include CDX attachments and match the pharma‑related classes above. Use the index to browse the weekly files.
